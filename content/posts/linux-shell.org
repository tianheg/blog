查看用户的默认 Shell：

#+BEGIN_SRC sh
    $ grep user /etc/passwd
    user:x:1000:1000::/home/user:/bin/bash
    # 或者
    $ ps -p $$
    PID TTY          TIME CMD
    22572 pts/1    00:00:00 bash
    # 或者
    $ echo "$0"
    bash
#+END_SRC

查看系统可用 Shell：

#+BEGIN_SRC sh
    $ cat /etc/shells
    # /etc/shells: valid login shells
    /bin/sh
    /bin/bash
    /usr/bin/bash
    /bin/rbash
    /usr/bin/rbash
    /usr/bin/sh
    /bin/dash
    /usr/bin/dash
    /usr/bin/tmux
    /usr/bin/screen
#+END_SRC

通过 =type -a bash= 找到 bash 的所有位置。

** =chsh=

更改当前用户 Shell

#+BEGIN_EXAMPLE
    chsh -s /bin/zsh
    echo $SHELL
#+END_EXAMPLE

列出所有 Shell

#+BEGIN_EXAMPLE
    chsh -l
#+END_EXAMPLE

** =useradd=

新建用户（有家目录）

refer [[https://wiki.archlinux.org/title/Users_and_groups]]

#+BEGIN_EXAMPLE
    useradd -m example
    passwd example
#+END_EXAMPLE

** =adduser=

=adduser= provides a high level interface for adding new users, and
=useradd= provides a low level interface.
