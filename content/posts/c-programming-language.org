* 声明变量

- int
- float
- char 声明单字符

变量声明后，变量对应的值可被更改。

** 格式化输出

- %d 输出整数
- %f 输出小数，%.1f 输出一位小数
- %c 输出单个字母

例子：

#+BEGIN_SRC c
int studentId = 15;
int studentAge = 23;
float studentFee = 45.6;
char studentGrade = 'C';

printf("Student id: %d\n", studentId);
printf("Student age: %d\n", studentAge);
printf("Student fee: %f\n", studentFee);
printf("Student grade: %c", studentGrade);
#+END_SRC

* 数据类型

- 整型 2/4 字节 %d

- 双精度浮点 8 字节 %lf

还有个 %s 用于字符串。

科学记数法： =34e2= ， =23E5= 。

小数精度设置： =%.1f=

占用内存大小使用 sizeof 查看： =sizeof(variable)=
