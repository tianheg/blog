#+TITLE: CSS 取代 br
#+DATE: <2022-05-22 Sun>
#+TAGS[]: 技术 CSS Hugo HTML

在 Hugo 中添加一个 shortcode：

#+BEGIN_EXAMPLE
    <div style="display: block;margin-bottom: 3em"></div>
#+END_EXAMPLE

在 Hugo 文章中使用：

#+BEGIN_EXAMPLE
    {{</* br */>}}
#+END_EXAMPLE

测试：

#+BEGIN_EXAMPLE
    test{{</* br */>}}test
#+END_EXAMPLE

test{{< br >}}test
