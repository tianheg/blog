## 例子

- <https://yihui.org/cn/>
- <https://io-oi.me/>

## 实现

在 Hugo 博客的博客单页面（single.html）中，修改 `{{ .Content }}` 为

```html
{{ $content := .Content }}
{{ $content = replaceRE `<a href="(https?://.+)">` `<a href="$1" target="_blank" rel="noopener">` $content | safeHTML }}
{{ $content | safeHTML }}
```

这样就可以实现「内链当前标签页打开，外链新标签页打开」。这可能是应用了「正则表达式」，我还不太了解。Hugo 的语法我还不了解。
