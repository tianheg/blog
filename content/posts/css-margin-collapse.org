#+TITLE: CSS 外边距(margin)折叠
#+DATE: <2023-03-20 Mon 08:51>
#+TAGS[]: 技术 CSS

{{< feynman >}}

https://www.w3schools.com/Css/css_margin_collapse.asp

上下外边距有时会折叠成一个。新的外边距的大小，取决于两者间最大的那一个。

左右外边距不发生折叠，仅发生在顶部和底部。

例子：

#+BEGIN_SRC html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
#+END_SRC

#+BEGIN_SRC css
h1 {
  margin: 0 0 50px 0;
}
h2 {
  margin: 20px 0 0 0;
}
#+END_SRC
