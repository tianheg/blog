* 深圳新能源硬件工程师求职路线图（26年7月-27年3月）

基于"目标定义→时间管理→坚持半年"框架。假设每周 *~8小时* 可投入，从7月开始到1月达到可投递状态。

*** 市场数据

- 深圳新能源初级硬件薪酬（2025-2026）：应届/1年经验 *13-18k·13薪*（约17-23万/年）
- JD高频技能：Altium Designer / KiCad、模数电路基础（op-amp/MCU最小系统）、有项目/竞赛经历（"电赛优先"反复出现）、会用示波器/万用表/烙铁
- 区域分布：南山16.5%、宝安14.3%、龙华12.1%、光明（BMS集中）
- 主要雇主：BYD（坪山/宝龙）、欣旺达（光明）、汇川技术（南山/光明）、台达/群光电能
- 加班程度：新能源硬件普遍大小周或单休，BMS尤甚

*** 第零步（现在）：方向自评

先确定方向。三个方向自评5问（每题1-5分）：

| 问题 | BMS | 电力电子 | 电源/充电桩 |
|------|-----|---------|----------|
| 我对电池化学/充放电曲线有基础认知 | | | |
| 我对电感/磁性元件/变压器不陌生 | | | |
| 我能说出一种DC-DC拓扑的工作原理 | | | |
| 我摸过示波器看过波形 | | | |
| 我对这个方向的日常工作在做什么有概念 | | | |

打分映射：BMS >= 15 选BMS（岗位最多）；电力电子 >= 15 选电力电子（门槛最高但竞争小）；都不达标选BMS（ADC/运放/CAN/充放电逻辑可迁移性强）。

*** 第一阶段（7-8月）：完成一块完整板子

目标：从原理图→PCB→打样→焊接→测试，走完一个完整闭环。一次就好。

主交付物：一块BMS单节电池保护板（或Buck变换器板）+ 测试报告

参考方案（BMS方向）：TI BQ29700（保护）+ 分压电阻网络采样 + STM32F103读电压电流通过UART输出 + 2层板KiCad + 过充/过放/过流保护

明确排除：不用刷Anki元器件参数，不用系统学Bode图/环路补偿

时间分配（~8h/周 × 8周）：

- 第1周：确定项目方案，读芯片datasheet，画系统框图（6-8h）
- 第2-3周：原理图设计（KiCad），边画边补电路知识（12-16h）
- 第4-5周：PCB布局布线，投板前检查（12-16h）
- 第6周：等板子，读论文/BMS白皮书（2-4h）
- 第7周：焊接+调试（8-10h）
- 第8周：测试+写测试报告+拍照留档（6-8h）

*Check-in gate ①（8月底）：* 板子能工作了吗？不工作但找到了原因也算过关；完全没进展且每周投入 < 3h → stop，评估时间问题。

*** 第二阶段（9-10月）：行业聚焦

目标：具备一个方向的面试应答能力 + 第二个项目

路径A：BMS（推荐）
- 学习TI BMS系列参考设计文档（BQ系列）
- 把单节板扩展成3-4串BMS原型（BQ76930方案）
- 学会CAN总线通信（TJA1050收发器）
- 面试高频题：passive vs active balancing、差分ADC采样原因、绝缘检测方法、BMS状态机

路径B：电力电子（电感/运放基础好的话）
- Fundamentals of Power Electronics (Erickson) 前三章
- 做一个12V→3.3V Buck转换器，测电压纹波、开关节点波形
- 面试高频题：Buck电感电流CCM/DCM画波形、反馈环路补偿、MOSFET开关损耗

时间分配：

- 第1-2周：读2-3份技术参考文档/白皮书，做笔记（12-16h）
- 第3-5周：第二个项目原理图+PCB（12-16h）
- 第6周：等板子，补面试题库（4-6h）
- 第7-8周：焊接调试+补面试题（12-16h）

*Check-in gate ②（10月底）：* 有手感吗？能完整说出技术细节就算过关。如果3周不动、每次做都抗拒 → 考虑换方向或承认硬件不适合。FAE/嵌入式软件也是退路。

*** 第三阶段（11-12月）：求职准备 + 开始投

交付物：

投递节奏：

*Check-in gate ③（12月底）：* >= 2家面试 → 冲刺；0面试 → 改简历，目标调低到中小公司

*** 第四阶段（1-3月）：冲刺

- 每面完一家当天写面经（被问倒的问题就是下阶段学习清单）
- 保持每周做一点电路（新的测试/测量），让手不生
- 2月还没offer → 主动去中小公司（宝安/龙华）扫街式投递

*** 退出信号

这不是"必须完成的计划"。三个stop/pivot点：

1. *8月底板子没焊出来也没找到原因* → 动手硬件不适合，优先找嵌入式软件/FAE/技术支持岗
2. *10月底第二个项目没有明显推进* → 退一步从FAE或测试岗切入，公司内部转研发
3. *12月底0面试* → 简历有问题/目标定错了/行业景气度不对，评估是否延期到27年3月窗口
