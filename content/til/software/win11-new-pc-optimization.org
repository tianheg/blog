* Win11 新机优化（含三方资料汇总）

新机到手后的系统清理和优化步骤，汇总自原文及三篇参考文章。

**参考来源：**
- https://blog.osyb.cn/posts/wz-33/
- https://sspai.com/post/92064
- https://winxuan.github.io/posts/win11-new-system-optimization/

---

## 一、初始化与账户

- **OOBE 绕过联网：** 联网界面按 `Shift+F10` → `oobe\BypassNRO` → 重启后出现"我没有 Internet 连接"选项 → 配置离线账户（避免微软邮箱名被截取为用户名导致路径乱码）
- 隐私设置全部关闭：广告ID、位置、诊断数据等

## 二、卸载预装与关闭广告

- 卸载厂商全家桶（McAfee 等），Windows Defender 足够日常使用
- 关闭开始菜单广告：设置→个性化→开始，关闭"显示建议的应用"
- 关闭通知与建议：设置→系统→通知，关闭"使用 Windows 时获取提示和建议"
- 搜索栏：设置→隐私和安全性→搜索权限→关闭"显示搜索要点"

## 三、系统底层优化

- **视觉效果：** 搜索"查看高级系统设置"→性能→调整为最佳性能（或保留平滑字体边缘/缩略图/视觉样式三项）
- **关闭传递优化：** 设置→Windows 更新→高级选项→传递优化→关闭"允许从其他电脑下载"
- **关闭非必要开机自启项**
- **文件资源管理器→查看→选项→常规→打开方式改为"此电脑"**
- **显示文件扩展名与隐藏文件：** 文件资源管理器→查看→勾选"文件扩展名""隐藏项目"
- **电源计划：** 控制面板→电源选项→笔记本选"平衡"，台式机选"高性能"。还可开启"卓越性能模式"（管理员CMD: `powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61`）
- **关闭虚拟化安全性（提升游戏帧数）：** 管理员CMD: `bcdedit /set hypervisorlaunchtype off` → 重启（不影响虚拟机使用）
- **关闭休眠释放C盘：** 管理员CMD: `powercfg -h off`（释放与内存等大的空间）

## 四、驱动与存储

- **显卡驱动：** NVIDIA/AMD/Intel 官网下载，不用第三方驱动软件
- **驱动管理（可选）：** 通过组策略（gpedit.msc → Windows 更新不包括驱动程序）或注册表禁止 Windows 更新自动装驱动，再从品牌官网手动装
- **更改新内容保存位置：** 设置→系统→存储→保存新内容的地方→改到D盘
- **开启存储感知：** 设为每天/每周，回收站超过30天自动清理
- **调整虚拟内存：** 控制面板→系统→高级系统设置→性能→设置→高级→虚拟内存，建议系统管理

## 五、个性化与使用习惯

- **任务栏：** 关闭小组件/聊天图标，搜索仅显示图标，任务栏左对齐，永远合并按钮
- **右键菜单恢复 Win10 风格（注册表）：** `HKCU\SOFTWARE\CLASSES\CLSID\` 新建键 `{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32` → 注销或重启资源管理器
- **关闭动画效果（可选）：** 设置→辅助功能→视觉效果→关闭"动画效果"；或高级系统设置→性能→关闭"对窗口内的控件和元素进行动画处理"
- **护眼：** 夜间模式（设置→系统→显示）
- **中文字体渲染：** 笔记本 PPI ≥ 172（16寸至少 2K），显示器建议 4K+ 缩放 200%
- **HDR：** 屏幕至少 HDR 600（OLED true black 400 除外），用 HDR Display Calibration 校准，N 卡用 RTX HDR

## 六、安全与备份

- **BitLocker：** 推荐对系统盘开启（Windows 最无感的加密方式），恢复密码建议打印保存
- **系统备份：** 控制面板→"备份和还原（Windows 7）"支持全量备份；Dism++ 支持增量备份
- **微软电脑管家：** 软件本身无问题（无广告），但会静默安装，直接卸载即可

## 七、推荐第三方工具

- **Scoop：** 包管理器（PowerShell 安装），配合 UniGetUI 统一管理
- **Everything：** 文件搜索神器，NTFS USN 日志检索，数秒搜 TB 级硬盘
- **PowerToys：** 微软官方工具集——FancyZones 窗口管理、Always on Top 置顶、PowerToys Run 快速搜索、File Locksmith 检查文件占用、Keyboard Manager 键位映射
- **ExplorerPatcher：** 任务栏/右键菜单调回 Win10 风格
- **PowerPlanSwitcher：** 系统托盘切换电源计划
- **其他：** PotPlayer（视频）、Q-dir（多文件夹管理）、VMware（已免费）、VisualCppRedist AIO（运行库合集）

## ⚠️ 避坑

- 不要用 XX 管家/大师——本身就是最大的广告源
- 不要盲目精简系统，保持安全更新
- Edge 自带 Chrome 内核，可使用 Chrome 插件，适合国人的微软账户同步
