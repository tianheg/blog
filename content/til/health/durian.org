#+TITLE: 榴莲的营养与好处
#+DATE: <2026-07-18 Sat>
#+TAGS[]: 水果 营养

* 核心营养

| 成分 | 含量/特点 |
|------|----------|
| 热量 | 高，约 147 大卡/100g |
| 碳水 | 主要为简单糖和淀粉 |
| 脂肪 | 富含健康脂肪 |
| 维生素 C | 抗氧化 |
| B 族维生素 | B1、B6、叶酸 |
| 矿物质 | 钾、锰、镁、铁 |
| 膳食纤维 | 促进肠道蠕动 |
| 多酚 | 抗氧化 |

* 具体益处

- *抗氧化*：多酚类物质清除自由基
- *补气血*：中医认为性热，适合体质虚寒、手脚冰凉者
- *缓解疲劳*：高糖分 + 钾，运动后恢复快
- *改善便秘*：纤维含量高，促进肠道蠕动
- *提供能量*：高热量高碳水，适合快速补充

* 注意事项

- 热量和糖分都高，糖尿病/减脂期慎吃
- 性热，吃多容易上火（口腔溃疡、便秘）
- 不要和酒精同食（可能影响代谢）
- 肾功能不全者注意钾摄入
- 建议每次 1--2 瓣（约 100--200g）
