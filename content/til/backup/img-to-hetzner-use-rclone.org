* 用Rclone备份图片到Hetzner StorageBox

[[https://docs.hetzner.com/storage/storage-box/access/access-ssh-rsync-borg/#rclone]]

基础配置很简单，复杂起来也挺头痛。

=~/.config/rclone/rclone.conf= ：

#+BEGIN_SRC conf
[storagebox]
type = sftp
host = uXXXXX.your-storagebox.de
user = uXXXXX
port = 23
pass = <obscured-password>
#+END_SRC
