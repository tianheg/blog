#+TITLE: 低压差线性稳压器 (LDO)
#+DATE: <2026-07-15 Wed>

LDO (Low Dropout Regulator) 是一种线性稳压器，用于将较高输入电压稳定转换为较低输出电压，允许输入-输出压差极小（~100mV~500mV）。相比传统线性稳压器（如7805需≥2V压差），LDO 在电池供电和噪声敏感场景有显著优势。

**核心特点：**

- **低压差：** 输入-输出电压差可低至 100mV，例如 Vin=3.4V 仍可稳定输出 3.3V。
- **低噪声：** 无开关动作，输出纹波极小，适合传感器/ADC-DAC/射频模块。
- **简单外围：** 仅需输入/输出电容，无电感。

**工作原理：** 功率晶体管（MOSFET/BJT）+ 误差放大器动态调节导通程度——输出下降时增大导通，反之减小。晶体管工作在线性区，压差越小功耗越低。

**典型应用：**

- 电池供电设备（手机、可穿戴），低压差延长续航
- 噪声敏感电路（音频芯片、精密测量）
- 开关电源后级滤波

**关键参数：**

| 参数 | 说明 |
|------|------|
| Dropout Voltage | 维持稳压的最小 Vin-Vout 差 |
| 静态电流 (Iq) | 空载自身消耗，影响电池寿命 |
| PSRR | 输入噪声抑制能力（高频可能下降） |
| 热阻 (Rθ) | 决定散热能力，影响最大输出电流 |

**优缺点：**

| 优点 | 缺点 |
|------|------|
| 低噪声、无 EMI | 效率较低（压差×电流=发热） |
| 电路简单、成本低 | 大压差时发热严重，需散热 |
| 快速响应负载变化 | 输出电流能力弱于开关稳压器 |

**示例型号：**

- TPS7A4700 (TI)：超低噪声（4.7µV[RMS]），医疗设备
- XC6206 (Torex)：小封装、低 Iq，IoT 设备
- MIC5205 (Microchip)：高 PSRR，射频模块

**与 DC-DC 对比：**

- LDO：小压差/低噪声首选，但效率随压差线性下降（3.6V→3.3V ≈92%，5V→3.3V 仅 66%）
- DC-DC：效率>90%、噪声大、电路复杂，适合大压差/高功率
