#+TITLE: Emacs 下将 Org 导出为 pdf
#+DATE: <2021-12-27 Mon>
#+HUGO_TAGS: Orgmode 技术

使用系统：Arch Linux

起初想安装 texlive-core，texlive-latexextra 等包，通过 Tex 导出，后来通过这篇文章[fn:pandoc]找到更快速的办法——Pandoc。它是一个强大的文本格式转换软件，在 Emacs 中使用可以安装 pandoc-mode[fn:pandoc-mode] 开启。通过快捷键 =M-x pandoc-convert-to-pdf= 即可导出。如果却少某些包，直接安装即可。

[fn:pandoc-mode] https://joostkremers.github.io/pandoc-mode/
[fn:pandoc] https://linuxhint.com/documents_emacs_org_mode/
