** 中文标点符号
   :PROPERTIES:
   :CUSTOM_ID: 中文标点符号
   :END:
| 符号             | 名称                                                                                             |
|------------------+--------------------------------------------------------------------------------------------------|
| 。               | 句号                                                                                             |
| ，               | 逗号                                                                                             |
| 、               | 顿号                                                                                             |
| ；               | 分号                                                                                             |
| ：               | 冒号                                                                                             |
| 「」『』“”''     | 直角引号、弯引号                                                                                 |
| （）［］〔〕【】 | 圆括号、方括号、六角括号、方头括号                                                               |
| ------           | 破折号                                                                                           |
| ......           | 省略号                                                                                           |
| －-～            | 连接号（一字线：占一个字的位置；短横线：占半个字的位置；浪纹线：占一个字的位置。它们都上下居中） |
| ·．              | 间隔号                                                                                           |
| 《》〈〉﹏﹏     | 书名号（双书名号、单书名号、浪线书名号）                                                         |
| ＿＿             | 专名号                                                                                           |
| .                | 着重号                                                                                           |
| /                | 分隔号                                                                                           |
