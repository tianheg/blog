Type, class, and ID selectors

类型选择器实际上就是标签（或元素）选择器

通用选择器就是*。如果单独使用，指代文档中的所有 DOM 节点；如果和选择符一起放在某元素之后，那么它就指代该元素下的所有内容

* 的一个用途就是让选择器更易读，更明显地发现他们的功能

注意：ID 比其他选择器具有更高的特异性。大多数情况下，更倾向于为元素添加类名

Attribute selectors

  • [attr] - a[title]
  • [attr=value] - a[href="https://example.com"]
  • [attr~=value] - p[class~="special"]
  • [attr|=value] - div[lang|="zh"]
  • [attr^=value] - li[class^="box-"]
  • [attr$=value] - li[class$="-box"]
  • [attr*=value] - li[class*="box"]

属性和属性值是大小写敏感性的，如果想改成不敏感，要在闭合中括号前加上 i

Pseudo-classes and pseudo-elements

伪类选择的是元素的特定状态

如果这样写 CSS


        article :first-child {
          ...
        }

就会选中 article 元素子代的第一个元素

例子：

:first-child Veggies es bonus vobis, proinde vos postulo essum magis kohlrabi welsh onion daikon amaranth tatsoi tomatillo melon azuki bean garlic.

:nth-child(2) Gumbo beet greens corn soko endive gumbo gourd. Parsley shallot courgette tatsoi pea sprouts fava bean collard greens dandelion okra wakame tomato. Dandelion cucumber earthnut pea peanut soko zucchini.

:last-child Gumbo beet greens corn soko endive gumbo gourd. Parsley shallot courgette tatsoi pea sprouts fava bean collard greens dandelion okra wakame tomato. Dandelion cucumber earthnut pea peanut soko zucchini.

伪元素：

它就像直接操作 HTML 元素，它是以 :: 开头的，但是为了兼容早期的 CSS 实现，有些伪元素也可以用 : 开头

::first-line ⬇️

可以将伪类和伪元素结合在一起使用


        article p:first-child::first-line {
          ...
        }
